To run the SAITS and BRITS models run the python file "SAITS_&_BRITS_Benchmarking_Implementation.ipynb"

This file contains both SAITS and BRITS implementations.
When testing the trained model the highlighted sections in the config files for SAITS and BRITS need to be updated.
SAITS config file: PhysioNet2012_SAITS_best.ini
BRITS config file: PhysioNet2012_BRITS_best.ini

Config file can be found in the config folder
